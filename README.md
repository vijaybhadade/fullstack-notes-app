# Secure Notes App

## Features
- User Registration & Login
- JWT Authentication
- Password Hashing with bcrypt
- Create, Update, Delete Notes
- Search Notes
- Protected Routes
- PostgreSQL Database
- Responsive Dashboard UI

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: PostgreSQL
- Authentication: JWT, bcrypt
